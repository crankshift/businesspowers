# businesspowers / ua

Плагін `ua` монорепо `businesspowers` — колекція субагентів і скілів для Claude Code і Codex на підтримку **ведення ФОП** (фізичної особи-підприємця) та **податкових питань фізичної особи** в Україні.

> ⚠️ **Не є податковою консультацією.** Інструмент допомагає орієнтуватись у процедурах, ставках і звітності, але остаточні розрахунки і подання мають пройти через бухгалтера або податкового консультанта (або, принаймні, через звірку з актуальним текстом Податкового кодексу та кабінетом платника на tax.gov.ua).

## Для кого

- **ФОП** 1, 2, 3, 4 груп єдиного податку та на загальній системі.
- Айтівці, фрілансери, самозайняті на групі 2 або 3 з КВЕД 62.01, 63.11, 62.02.
- **Фізичні особи**, які:
  - отримують іноземні дивіденди (S&P 500, американські акції);
  - працюють з IBKR, Freedom Finance, Wise, Revolut;
  - продавали акції або ETF у 2024–2025;
  - мають крипто активність (BTC, ETH, стейблкоіни);
  - отримали спадщину, подарунки, виграші.
- **Бухгалтери**, які супроводжують кількох ФОП і періодично мають різнотипні кейси (закриття, зміна групи, додавання КВЕД, звітність за квартал/рік).
- **ФОП-релоканти**, які розриваються між ФОП в Україні та JDG у Польщі (для JDG-частини — плагін `/pl:…`).

## Встановлення

```
/plugin marketplace add crankshift/businesspowers
/plugin install ua@businesspowers
/reload-plugins
```

Після цього всі агенти і скіли плагіна виходять з префіксом `/ua:…`.

Щоб перевірити:

```
/agents
```

Мають з'явитись `ua:fop-registrator`, `ua:tax-system-advisor`, `ua:fop-tax-calculator` і т.д.

## Встановлення в Codex

Codex ID цього плагіна: `business-ua`. Claude Code ID лишається `ua`.

З marketplace GitHub:

```bash
codex plugin marketplace add crankshift/businesspowers
```

Локально з checkout репозиторію:

```bash
cd /path/to/businesspowers
codex plugin marketplace add .
```

Після додавання marketplace увімкніть `business-ua` у Codex plugin UI / marketplace flow. Codex читає інструкції з `AGENTS.md`; Claude Code читає `CLAUDE.md`.

Codex custom-agent files are included in `.codex/agents/`. They are generated from `agents/*.md`, so update the Markdown source first and run the repo-level converter/validator before release.

## Каталог агентів

### Блок ФОП — життєвий цикл

| Агент | Призначення |
|---|---|
| `fop-registrator` | Реєстрація ФОП — через Дію, ЦНАП чи нотаріуса. Вибір КВЕД, заява на спрощену систему, отримання виписки з ЄДР, підключення до податкової, відкриття банківського рахунку (mono, privat, sense). Крок-за-кроком. |
| `tax-system-advisor` | Допомога у виборі групи єдиного податку (1, 2, 3, 4) vs загальної системи. Порівнює обороти, КВЕД, дозволені види діяльності, ставки, обов'язок РРО, ПДВ. Видає рекомендацію з обґрунтуванням. |
| `fop-tax-calculator` | Розрахунок єдиного податку (ставки на дату), ЄСВ, ПДВ (якщо застосовно), штрафів за прострочення. Враховує воєнні пільги (періоди звільнення від ЄСВ, пільги для мобілізованих). |
| `fop-reporting-agent` | Звітність ФОП: податкова декларація платника ЄП (квартальна для групи 3; річна для груп 1, 2, 4), звіт з ЄСВ (річний — додаток 1 ДС), 1-ДФ / 4-ДФ (якщо є найманці чи виплати фізособам). Терміни, форми, електронний кабінет. |
| `fop-closer` | Закриття ФОП: подача форми 8-ОПП через Дію / ЦНАП, погашення боргів, фінальна звітність, зняття з обліку в ПФУ / податковій. Типові граблі: неподана фінальна звітність, борги по ЄСВ після закриття. |

### Блок фізособа — податки за межами ФОП

| Агент | Призначення |
|---|---|
| `physical-person-tax-advisor` | Загальні консультації по податкам фізичної особи: декларація про майновий стан і доходи (ст. 179 ПКУ), коли подавати (до 1 травня), коли не подавати, хто звільнений. Спадщина, подарунки, продаж майна. |
| `investment-tax-agent` | Декларування інвестицій і пасивного доходу: американські акції через IBKR / Freedom24 / Wise / Revolut, дивіденди (ПДФО 9% для зареєстрованих в Україні, 18% для іноземних — або пільгові ставки за конвенціями), інвестиційний прибуток, крипто (поки що через загальні норми), роялті. Курс НБУ, W-8BEN, форма 1042-S. |

### Блок інвойсинг — операційна робота

| Агент | Призначення |
|---|---|
| `invoice-manager` | Виставлення рахунків на оплату, актів наданих послуг (UAH B2B), англомовних Invoice (USD/EUR/GBP для клієнтів США/ЄС), ПН в ЄРПН для ФОП-платників ПДВ, коригуючі розрахунки (РК). Реквізити SWIFT/IBAN, нумерація, робота з mono/privat24/sense/fuib. |
| `invoice-analyzer` | Реєстр інвойсів (оплачені/очікує/прострочено), aging report, cash-flow прогноз, клієнтська аналітика (TOP-10), парсинг банківських виписок, звірка з декларацією платника ЄП, перевірка контрагентів (реєстри ДПС / ПДВ / ризиковість), пеня за ст. 625 ЦК, претензійні листи. |

## Приклади сценаріїв

### «Я хочу відкрити ФОП для IT-фрілансу»

```
Я працюю remote розробником, хочу відкрити ФОП, дохід ~200 000 грн/міс, клієнти з США.
Яку групу обрати і як реєструватись?
```

Claude викличе `ua:tax-system-advisor` → аналіз: група 3, ставка 5%, без ПДВ; КВЕД 62.01; далі `ua:fop-registrator` → крок-за-кроком через Дію; далі `ua:calculating-edynyi-podatok` → орієнтовний розрахунок податків.

### «Я забув подати декларацію за минулий рік»

```
Я ФОП група 3, пропустив квартальну декларацію ЄП за IV кв. 2025 р.
Які штрафи і як швидко виправити?
```

Claude викличе `ua:fop-reporting-agent` → посилання на ст. 300 ПКУ (штраф за неподання), ст. 123 ПКУ (штраф за несплату), алгоритм: подати з позначкою «уточнююча», розрахувати пеню, сплатити.

### «Я купив S&P 500 через IBKR, що робити з податками?»

```
Купив акції Apple через Interactive Brokers у 2025, отримав дивіденди $120,
США утримало 15% за W-8BEN. Як відобразити в декларації?
```

Claude викличе `ua:investment-tax-agent` → нарахування дивіденду за курсом НБУ на дату зарахування; ст. 170.11 ПКУ (пасивний дохід з-за кордону) — ПДФО 9% (для іноземних дивідендів зі зменшеною ставкою) + ВЗ 5% + застосувати США утримання за конвенцією. Далі `ua:declaring-investments` → конкретне заповнення ряд. 10.4 додатка до декларації про майновий стан.

### «Чи відкривати ФОП на групі 2 для салону краси?»

```
Відкриваю салон краси у Львові, 3 майстра + я, обіг очікую 150 000 грн/міс.
Група 2 підходить?
```

Claude викличе `ua:tax-system-advisor` → група 2 дозволена для послуг населенню з обмеженнями по найманим (до 10 осіб) і продажу юрособам (заборонено продавати юрособам на ЄП, тільки населенню і іншим ФОП на ЄП); ставка — до 20% МЗП на 2026; РРО не обов'язковий до певного ліміту; КВЕД 96.02.

## Нормативна база

Основні акти, на які спирається плагін:

- **Податковий кодекс України** (`2755-17`) — гл. 1 розд. XIV (єдиний податок, ст. 291–300), розд. IV (ПДФО, ст. 162–179).
- **ЗУ «Про збір та облік єдиного внеску на загальнообов'язкове державне соціальне страхування»** (`2464-17`) — ЄСВ.
- **ЗУ «Про Державний бюджет України»** на відповідний рік — визначає розмір мінімальної зарплати і прожиткового мінімуму, від яких рахуються ставки.
- **Наказ МФУ № 308 від 19.06.2015** — форма заяви про застосування спрощеної системи.
- **Наказ МФУ № 859 від 02.10.2015** — форма декларації про майновий стан (для фізосіб).
- **Наказ МФУ № 1118 від 19.09.2016** — форма звіту з ЄСВ.
- Під час воєнного стану — підзаконні акти про пільги (звільнення від ЄСВ, перехід на 2%, відсутність штрафів).

Див. повний список у `CLAUDE.md` → секція «Ключові ресурси».

## Відмова від відповідальності

Цей плагін — **допоміжний інструмент, не податкова консультація**. Рішення про реєстрацію, вибір групи, розмір сплати податків, форму і зміст декларації — приймає **платник податків особисто** (або його бухгалтер / податковий консультант). Ставки, порогові значення і форми міняються кожного року (а під час воєнного стану — і частіше). Перед будь-яким поданням — звірка з чинним текстом Податкового кодексу і актуальними роз'ясненнями ДПС.

Авторам плагіна і Anthropic не належить відповідальність за наслідки використання виданих агентами рекомендацій.
