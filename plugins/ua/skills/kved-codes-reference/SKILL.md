# kved-codes-reference

Класифікатор видів економічної діяльності (**КВЕД**) — обов'язковий елемент реєстрації ФОП. При реєстрації платник обирає один **основний** КВЕД і будь-яку кількість **додаткових**.

**Важливо:** при роботі на єдиному податку можна здійснювати діяльність **лише за КВЕД, зазначеними в Реєстрі платників ЄП**. Якщо дохід отримано за КВЕД, не зазначеним у реєстрі, → оподатковується за **подвійною ставкою 15% _(fallback; станом на 01.01.2026)_ ПДФО** (п. 293.4.2 ПКУ).

## Джерела

- Офіційний класифікатор: [kved.ukrstat.gov.ua](https://kved.ukrstat.gov.ua) — Державна служба статистики.
- КВЕД-2010 — діюча версія в Україні (гармонізована з NACE rev. 2 ЄС).
- Заборонені для ЄП КВЕД — **п. 291.5 ст. 291 ПКУ** (`zakon.rada.gov.ua/laws/show/2755-17`).

## Параметри — перехресне посилання

> Актуальні значення — отримувати через канонічні скіли.
> Якщо скіл не викликаний, використати fallback нижче; попередити: «⚠ Значення [параметр] використано зі станом на [дата].»

| Параметр | Канонічний скіл | Fallback |
|---|---|---|
| Подвійна ставка 15% / ліміти ЄП | `calculating-edynyi-podatok` | 15% _(01.01.2026)_ |

## Структура коду

```
XX.XX.X

Перший рівень — клас (XX) — наприклад, 62 (комп'ютерне програмування)
Другий — підклас (XX.XX) — наприклад, 62.01 (комп'ютерне програмування)
Третій — групування (XX.XX.X) — рідко вживається
```

При реєстрації в Дії / ЄДР зазвичай пишуть до другого рівня: 62.01, 47.91, 96.02.

## Заборонені для ЄП види діяльності (п. 291.5 ПКУ)

### 291.5.1 — діяльність, що підлягає обов'язковому ліцензуванню і при цьому не дозволена для ЄП

- **Азартні ігри, лотереї (крім дистриб'юторів державних лотерей для ФО)**.
- **Обмін іноземної валюти**.
- **Виробництво, експорт, імпорт, продаж підакцизних товарів** (крім роздрібного продажу паливно-мастильних матеріалів у ємностях до 20 л; крім діяльності ФОП з продажу пива, столових вин і окремих видів у межах переліку).
- **Видобуток, реалізація корисних копалин** (крім корисних копалин місцевого значення).
- **Страхова діяльність, діяльність кредитних спілок, ломбардів, лізингових компаній** тощо.
- **Діяльність у сфері фінансового посередництва** (крім страхових агентів).
- **Діяльність з управління підприємствами**.
- **Діяльність поштового зв'язку (крім кур'єрської).**
- **Продаж предметів мистецтва, антикваріату, передметів колекціонування**.
- **Організація гастрольної діяльності (концертно-видовищних заходів)**.
- **Технічні випробування та дослідження**.
- **Аудиторська діяльність**.
- **Діяльність з продажу нерухомого майна, а також оренда і управління нерухомим майном** (ця група має винятки; загалом обмеження по частці / типу об'єктів).

### 291.5.2 — фізособи-підприємці, які здійснюють певні види діяльності:

- Здійснюють обмін валюти.
- Виробляють, експортують, імпортують, продають підакцизні товари (крім переліку).
- Виробляють, продають ювелірні вироби.

### 291.5.3 — нерезиденти

Нерезиденти (у розумінні ПКУ) не можуть бути платниками ЄП.

### 291.5.4 — платники, що мають податковий борг

Якщо на дату подання заяви — податковий борг (окрім безнадійного).

## Як обирати КВЕД

### Крок 1. Визначити основну діяльність

**Що будеш робити найчастіше / з чого буде найбільша частка доходу?** Цей КВЕД — основний.

Приклад: розробляєш ПЗ на замовлення → 62.01. Консультуєш + розробляєш — основний 62.01, додатковий 62.02.

### Крок 2. Додати додаткові

Все, що можеш потенційно робити:
- 62.01, 62.02, 62.09 (IT)
- 70.22 (консалтинг)
- 74.20 (фотографія — якщо робиш foto sessii як хобі, але є ризик: дохід з 74.20 → оподатковуй за ЄП)

### Крок 3. Перевірити заборони для ЄП

**Прочитати кожен КВЕД у розрізі п. 291.5 ПКУ.** Якщо хоч один — заборонений → неможливо ЄП.

Наприклад, якщо додав 64.19 (грошове посередництво) → ЄП заблоковано для всієї діяльності, не тільки для цього КВЕД.

### Крок 4. Занести до Реєстру платників ЄП

Через заяву про застосування спрощеної системи (Наказ МФУ № 308). Заява містить список КВЕД, які будуть зареєстровані для ЄП.

**Якщо в процесі додаєш новий КВЕД:**
- Змінити в ЄДР (через Дію або ЦНАП — форма 11).
- Подати нову заяву про ЄП з оновленим списком.

## Типові помилки

| Помилка | Наслідок | Правильно |
|---|---|---|
| Обрав 64.19 (фінансове посередництво) «про запас» | ЄП заблоковано | Видалити заборонений КВЕД до подання |
| Отримав дохід за КВЕД, не зазначеним у Реєстрі ЄП | Оподаткування за подвійною ставкою 15% | Додати КВЕД заздалегідь (через Дію / ЦНАП) |
| Обрав 47.91 (інтернет-торгівля) без ПРРО | Штраф 100–300% від необлікованої суми | Поставити ПРРО або обмежити оборот нижче порога |
| Для адвокатської діяльності — КВЕД 69.10 + ЄП | ЄП заборонено для адвокатів | Тільки загальна система |
| Для ріелторської — КВЕД 68.31 без обмежень | Залежно від площі і частоти — може бути заборонено | Звіряти п. 291.5 ПКУ |

## Інтерпретації ДПС

- **ДПС видає роз'яснення** щодо конкретних КВЕД. Посилання — через `fetching-podatkova-gov-ua`.
- Пошук: `tax.gov.ua` → розділ «Податкові консультації» → за номером або ключовими словами.

## Приклад запису КВЕД при реєстрації через Дію

```
Основний:  62.01 Комп'ютерне програмування
Додаткові:
  62.02 Консультування з питань інформатизації
  62.09 Інша діяльність у сфері інформаційних технологій
  63.11 Оброблення даних, розміщення інформації на вебпорталах
  70.22 Консультування з питань комерційної діяльності
  74.10 Спеціалізована діяльність у сфері дизайну
  85.59 Інші види освіти, н. в. і. у.
  74.30 Надання послуг перекладу
```

Такий набір дає широку «IT + консалтинг + освіта» універсальність для фрілансера.

## Обмеження

- Скіл не охоплює повний каталог КВЕД — лише типові для ФОП.
- Для складних випадків (кілька видів діяльності з перекриттям) — перевіряти офіційний класифікатор.
- Окремі КВЕД підпадають під спеціальне ліцензування; перевіряти в кожному випадку.
