# fetching-podatkova-gov-ua

Скіл — робота з офіційним сайтом Державної податкової служби України (`tax.gov.ua`) та кабінетом платника (`cabinet.tax.gov.ua`).

## Головні ресурси

### 1. tax.gov.ua — головний портал ДПС

**Адреса:** [tax.gov.ua](https://tax.gov.ua).

Структура:
- **Головна** — новини, анонси.
- **Деталі податку** — роз'яснення за кожним податком (ПДВ, ПДФО, ЄП, ЄСВ).
- **Реєстр ІПК** — індивідуальні податкові консультації.
- **Форми звітності** — посилання на бланки.
- **Електронний кабінет** → перехід на cabinet.tax.gov.ua.

### 2. cabinet.tax.gov.ua — кабінет платника

**Адреса:** [cabinet.tax.gov.ua](https://cabinet.tax.gov.ua).

Функціонал:
- Подання звітності.
- Сплата податків (тільки перевірка, без платіжок — платіжки через банк).
- Перегляд стану розрахунків (заборгованість / переплата).
- Листування з ДПС.
- Реєстр платників ПДВ (перевірка контрагентів).
- Реєстр платників ЄП.

### 3. Реєстр ІПК (Індивідуальних Податкових Консультацій)

**Адреса:** [tax.gov.ua/individualni-podatkovi-konsultatsii](https://tax.gov.ua/individualni-podatkovi-konsultatsii) (або через пошук).

**Що таке ІПК:**
- Відповідь ДПС на запит конкретного платника щодо конкретної норми ПКУ.
- Обов'язкова тільки для платника, що звертався (ст. 52 ПКУ).
- Для інших — рекомендаційна.

**Формат:**
- Номер ІПК: `2024/25/1234567-ІПК`.
- Дата.
- Тема (за пошуковими словами).
- Повний текст відповіді ДПС.

**Пошук:**
- За номером.
- За темою / ключовими словами.
- За датою.

### 4. Open Data ДПС

- [opendata.tax.gov.ua](https://opendata.tax.gov.ua) — API та датасети.
- Реєстр платників ПДВ.
- Реєстр боржників.

## Типові сценарії

### Сценарій 1: Перевірка актуальної ставки ПДВ

**Запит:** «Яка ставка ПДВ на ліки на 22.04.2026?»

**Дія:**
1. Перевірити tax.gov.ua → «Деталі податку» → «ПДВ».
2. Альтернативно — ПКУ ст. 193 (скіл `fetching-zakon-rada` з `lawpowers`).
3. На 2026 — **7%** для ліків, медвиробів, **20%** стандарт, **0%** експорт.

### Сценарій 2: Пошук ІПК про крипто

**Запит:** «Як ДПС тлумачить оподаткування криптовалюти для фізосіб?»

**Дія:**
1. `tax.gov.ua` → пошук «віртуальні активи» / «криптовалюта».
2. ІПК від 2023–2025 років — тлумачення через ст. 170.2 ПКУ (інвестиційний прибуток).
3. **Обережність:** ІПК від 2020 року могла бути відмінена або інша позиція з 2023.

### Сценарій 3: Перевірка статусу платника ПДВ контрагента

**Запит:** «Чи є ТОВ "Альфа" (код 12345678) платником ПДВ?»

**Дія:**
1. `cabinet.tax.gov.ua` → «Реєстр платників ПДВ».
2. Альтернативно — `tax.gov.ua/reyestry` → реєстри.
3. Ввести ЄДРПОУ → перевірка.

### Сценарій 4: Подача декларації

**Запит:** «Як подати квартальну декларацію ЄП?»

**Дія:**
1. Зайти на `cabinet.tax.gov.ua`.
2. Авторизація КЕПом.
3. «Звітність» → «Подати звіт» → форма «Податкова декларація платника єдиного податку».
4. Вибрати період (квартал, рік).
5. Заповнити.
6. Підписати КЕПом → надіслати.
7. Квитанція №1 (прийом) + №2 (обробка).

### Сценарій 5: Стан розрахунків

**Запит:** «Чи є у мене борг з ЄП за минулий рік?»

**Дія:**
1. Авторизація в `cabinet.tax.gov.ua`.
2. «Стан розрахунків з бюджетом».
3. Переглянути по кожному податку:
   - Нараховано.
   - Сплачено.
   - Залишок (борг / переплата).

## Реєстри, доступні в кабінеті

| Реєстр | Призначення |
|---|---|
| Реєстр платників ПДВ | Перевірка статусу ПДВ контрагентів |
| Реєстр платників ЄП | Перевірка групи ЄП, видів діяльності |
| Реєстр ризикових платників | Перевірка контрагента на ризики (блокування податкових накладних) |
| Реєстр виписаних ПН | Перегляд своїх податкових накладних у ЄРПН |
| Реєстр платників ПДФО | Рідко використовується |

## Контроль над податковими накладними (ЄРПН)

**Система блокування (система моніторингу критеріїв ризиків, СМКОР):**

- При реєстрації ПН система автоматично перевіряє критерії ризиків.
- Якщо ПН «заблокована» — треба розблокувати (через подання пояснень і документів).
- Моніторинг — обов'язкова частина життя платника ПДВ.

## Офіційні роз'яснення ДПС

**Типи:**

1. **ІПК** — для конкретного платника (за заявою).
2. **Узагальнюючі податкові консультації** — для невизначеного кола платників (ст. 52-1 ПКУ). Публікуються на tax.gov.ua.
3. **Листи роз'яснення** — рідше, але бувають.
4. **Оглядові листи** (ОК, ОП) — роз'яснення великих новацій.

**Ієрархія довіри:**
1. **ПКУ** (закон) — завжди головне.
2. **Узагальнюючі ІПК** — застосовуються до всіх.
3. **ІПК для конкретного платника** — обов'язкові для нього.
4. **Листи** — не мають статусу НПА, але відображають поточну позицію ДПС.
5. **Роз'яснення на форумах tax.gov.ua** — додатково, не обов'язкові.

**Важливо:** роз'яснення ДПС **не можуть суперечити ПКУ**. Якщо ДПС дає ІПК з позицією, яка виходить за межі закону — суди не підтримують (приклад — рішення ВС у справах про обмеження відрахувань).

## Частіші URL-паттерни

### Податковий кодекс

Немає на tax.gov.ua прямо (це — на zakon.rada.gov.ua). Але tax.gov.ua дає **виклади, інтерпретації, розділення за податками**.

### Форми звітності

`tax.gov.ua/baneryi/podatkova-zvitnist/` — бланки основних декларацій.

### Контакти ДПС

`tax.gov.ua/kontaktna-informatsiya/` — контакти центральної ДПС + регіональних.

### Кабінет ДПС за КЕПом

`cabinet.tax.gov.ua` — через Дія.Підпис / Ключ.України / Приватбанк / АЦСК.

## Нюанси кабінету

### Отримання квитанцій

- **Квитанція №1** — прийом звіту системою (не означає прийняття).
- **Квитанція №2** — обробка звіту (прийнято / відмовлено з причиною).
- Зберігати обидва як доказ подання.

### Відмови

- Найчастіші причини: помилки контрольних сум, неактуальна адреса платника, не прийнято попередню декларацію, технічні помилки XML.
- При відмові — виправити і подати знову.

### Листування з ДПС

- Розділ «Повідомлення» в кабінеті.
- ДПС може надсилати вимоги про надання пояснень.
- Відповідати треба у визначений термін (зазвичай 10 днів).

## Перевірка контрагентів

Ключовий функціонал: перед укладанням угоди з контрагентом (особливо для платника ПДВ) — перевіряти:

1. **ЄДРПОУ / РНОКПП** в реєстрі платників ПДВ.
2. **Статус платника** ЄП (в реєстрі платників ЄП).
3. **Наявність у реєстрі ризикових платників** (СМКОР).
4. **Діюча реєстрація** (не припинення).
5. **Діюча адреса** (витяг з ЄДР).

## Альтернативи кабінету

### Сторонні сервіси (для подання звітів)

- **M.E.Doc** — найпотужніша, платна.
- **Sonata** — популярна для ФОП.
- **Fredo Звіт** — безкоштовна.
- **1C:Бухгалтерія** — комплекс.

Всі подають через АРІ ДПС, результат тотожний.

### Мобільний додаток ДПС

- «ДПС: Податки в смартфоні» (Play / App Store).
- Базові функції: стан розрахунків, перегляд реєстрів, подача простих звітів.

## Обмеження

- `tax.gov.ua` **не** є джерелом норми права — норма у **ПКУ** (`zakon.rada.gov.ua`).
- Роз'яснення ДПС можуть змінюватись; перевіряти дату.
- ІПК для конкретного платника — не застосовується до інших.
- Кабінет може бути недоступний у період технічних робіт (зазвичай у вихідні).
- При недовірі до ДПС — перевіряти через Мінюст (`usr.minjust.gov.ua` для ЄДР).
