# declaring-investments

Скіл — як задекларувати інвестиційні доходи фізичної особи-резидента України в річній декларації про майновий стан і доходи.

## Ключові джерела

- **ПКУ** (`zakon.rada.gov.ua/laws/show/2755-17`):
  - Ст. 170.2 — інвестиційний прибуток.
  - Ст. 170.11 — іноземні доходи.
  - Ст. 167.5.4 — дивіденди іноземні 9%.
  - Ст. 167.5.2 — дивіденди українські 5%.
  - Ст. 164.4 — курс НБУ.
- **Наказ МФУ № 859** (`z1298-15`) — форма декларації.
- **Конвенції про уникнення подвійного оподаткування** — в кожному випадку з `zakon.rada.gov.ua`.

## Принцип

Декларація фізособи — **до 1 травня** наступного року (ст. 179 ПКУ).
Сплата — **до 1 серпня**.

### Що включити

1. **Інвестиційний прибуток (ст. 170.2)** — купівля-продаж акцій, ETF, облігацій, деривативів → додаток Ф1. Ставка **18% + 5% = 23%**.
2. **Дивіденди** — додаток Ф3. Ставки:
   - Українські від ТОВ на загальній: 5% + 5% ВЗ.
   - Іноземні: 9% + 5% ВЗ (з заліком утримання).
   - Українські від неплатника податку на прибуток / ІСІ: 18% + 5%.
3. **Проценти** (від іноземних облігацій) → додаток Ф3. 18% + 5%.
4. **Роялті** — 18% + 5%.

### Що НЕ включати

- Українські дивіденди / проценти, з яких агент утримав податок — нічого додавати (але можна показати як інформаційний).
- Продаж українських ОВДП — 0%.

## Робота з брокерами

### Interactive Brokers (IBKR)

**Що потрібно:**
- Звіт **Activity Statement** за рік (PDF, CSV або Flex Query).
- Інтерактивний Flex Query з полями: Trade Date, Symbol, Buy/Sell, Quantity, Price, Commission, Proceeds, Basis, Currency.
- Якщо дивіденди — звіт **Dividend Summary** + **1042-S** (для резидентів-нерезидентів США; містить утримання).

**Алгоритм:**

1. У IBKR Portal → Reports → Activity.
2. Обрати «Custom Date Range»: 1 січня – 31 грудня звітного року.
3. Формат: PDF + CSV для Excel.
4. Додатково — **Performance and Attribution** для перевірки.

**Поля для декларації:**

| Поле IBKR | Що означає | Використовується |
|---|---|---|
| Trades → Commission | Комісія брокера | Вартість придбання/продажу |
| Trades → Proceeds | Сума продажу (брутто) | Дохід |
| Trades → Basis | Ціна придбання | Вартість |
| Dividends | Брутто дивідендів | Дохід для додатка Ф3 |
| Withholding Tax | Утримано США | Залік (15% при W-8BEN) |

**Розрахунок фінрезультату:**

```
Прибуток_по_позиції = Proceeds − Basis − Комісія
```

FIFO застосовується автоматично в IBKR.

### Freedom24

**Звіт:** Меню → Звіти → «Брокерський звіт» (CSV/XLS).

Є готовий **«Звіт для ДПС України»** з готовим розрахунком в гривнях — але **перевіряти**, бо часто:
- Помилки в курсах НБУ (може брати не на дату, а середньорічний).
- Неправильне визначення FIFO для тих паперів, що купували кількома траншами.

### Wise Stocks

**Звіт:** Кабінет Wise → Stocks → Statements.

- Позиції, дивіденди, сплачені комісії.
- Валюта — USD, EUR, GBP за зонами.
- Курс НБУ застосовується **на дату зарахування** на рахунок.

### Revolut

- Аналогічно Wise.
- Виписки — у форматі CSV / PDF.
- Акції Revolut — через US custodian; можливе утримання за W-8BEN 15%.

### Українські брокери

- **Універ Капітал**, **Альфа-Капітал**, **Інтеркапітал**, **Сігма-Капітал**, **Art Capital**.
- Часто **є податковими агентами** — утримують при виплаті дивідендів і при закритті позицій з прибутком.
- Фізособа отримує готовий **податковий звіт** для додавання в декларацію.
- Багато українських брокерів працюють з ОВДП, які звільнені від ПДФО (0%) — окрема категорія.

## Заповнення додатка Ф1 (інвестиційний прибуток)

### Структура (спрощено, згідно Наказу МФУ № 859)

| Рядок | Зміст |
|---|---|
| 1.1 | Доходи від продажу інвестиційних активів |
| 1.2 | Витрати на придбання інвестиційних активів |
| 1.3 | Фінансовий результат (1.1 − 1.2) |
| 1.4 | Сума оподатковуваного інвестиційного прибутку |
| 1.5 | Нарахований ПДФО (18% × 1.4) |
| 1.6 | Нарахований ВЗ (5% × 1.4) |

### Сальдування

Всередині року сума прибутків і збитків сальдується:

```
Річний_фін_результат = Σ (Profit_операцій) − Σ (Loss_операцій)
```

**Якщо результат < 0 → податок = 0.** Перенос на наступний рік **заборонено**.

### Приклад

**2025 рік, IBKR:**
- AAPL: прибуток $2 000 × 41 = 82 000 грн.
- MSFT: збиток $800 × 41 = 32 800 грн.
- TSLA: прибуток $500 × 41 = 20 500 грн.

**Розрахунок:**
- Σ прибутків: 82 000 + 20 500 = 102 500.
- Σ збитків: 32 800.
- **Фін. результат: 69 700 грн.**
- ПДФО: 18% × 69 700 = **12 546 грн**.
- ВЗ: 5% × 69 700 = **3 485 грн**.
- **Разом: 16 031 грн.**

### Курс НБУ

Принципове правило (п. 164.4 ПКУ): **курс НБУ на дату операції**.

Приклад: купив AAPL 10 акцій по $180 = $1 800, курс НБУ на 15.02.2025 = 41,5 → **74 700 грн**.
Продав 15.08.2025 по $220 = $2 200, курс = 41,3 → **90 860 грн**.
Прибуток: 90 860 − 74 700 = **16 160 грн**.

(Не: 40 × 41 = 1 640.)

Скіл `converting-currency-nbu` — деталі про курси.

## Заповнення додатка Ф3 (іноземні доходи)

### Структура

| Рядок | Зміст |
|---|---|
| 2.1 | Вид доходу (дивіденди, проценти, роялті, заробіток) |
| 2.2 | Країна джерела |
| 2.3 | Сума в іноземній валюті |
| 2.4 | Курс НБУ на дату отримання |
| 2.5 | Сума в грн |
| 2.6 | Сума утримання в країні джерела |
| 2.7 | Нарахований український ПДФО |
| 2.8 | Залік іноземного утримання |
| 2.9 | ПДФО до сплати в Україну |
| 2.10 | ВЗ до сплати в Україну |

### Ставки ПДФО у рядку 2.7

- Іноземні дивіденди: **9%** × сума в грн.
- Інші іноземні доходи (проценти, роялті, зарплата): **18%**.

### Залік утримання (ст. 170.11.2)

```
Залік_в_грн = min(Утримання_в_грн, Нарахований_ПДФО)
ПДФО_до_сплати = Нарахований_ПДФО − Залік
ВЗ_до_сплати = 5% × Сума_в_грн (без заліку!)
```

### Приклад: Apple дивіденди

**Вхідні:**
- Apple Inc. (США), дивіденди $120, виплачені 15.05.2025.
- Утримано США 15% (за W-8BEN): $18.
- Курс НБУ на 15.05.2025: 41,5.

**Розрахунок:**

| Поле | Значення |
|---|---|
| Брутто в USD | 120 |
| Курс НБУ | 41,5 |
| Брутто в грн | 4 980 |
| Утримання США в USD | 18 |
| Утримання в грн | 747 |
| Нарахований ПДФО (9%) | 448,20 |
| Залік (min) | 448,20 |
| ПДФО до сплати в Україну | 0 |
| ВЗ до сплати | 5% × 4 980 = 249 |

**Разом до сплати в Україну: 249 грн.**

### Приклад: VWCE (Ірландія) дивіденди

**Вхідні:**
- VWCE (FTSE All-World UCITS ETF, Ірландія), дивіденди €150 (акумулюючі? Для акумулюючих — нічого не виплачується, для дистриб'юшн — виплата).
- Припустимо — дистриб'юшн: €150 отримано 20.07.2025.
- Ірландія утримала 0% (UCITS).
- Курс НБУ 43,5.

**Розрахунок:**
- Брутто в грн: 150 × 43,5 = 6 525.
- ПДФО 9%: 587,25.
- ВЗ 5%: 326,25.
- Разом: **913,50 грн**.

## W-8BEN для США

### Для чого

- Зменшити утримання США з 30% до 15% за конвенцією US-UA (ст. 10).

### Як отримати

- Брокер надає форму в онлайн-кабінеті при реєстрації.
- Заповнити:
  - Ім'я, прізвище (латиницею як у паспорті).
  - Резидентство: Ukraine.
  - Податкова адреса: в Україні.
  - РНОКПП — необов'язково, але можна внести.
  - Дата, підпис.
- Діє **3 роки**, після — оновити.

### Якщо не підписав

- США утримує **30%**.
- В Україні нарахування 9%; залік = 9%; зайві 21% **втрачені** (не повертаються ні в США, ні в Україні).

## Крипто — окрема історія

Скіл `declaring-crypto` — деталі.

Коротко:
- Купівля-продаж crypto → ст. 170.2 (інвестиційний прибуток, 18%+5%).
- Кожен обмін crypto/crypto — фактично реалізація → податок (на практиці часто ігнорують).
- Закон про оподаткування віртуальних активів (№ 10225) — **перевіряти статус ухвалення**.

## Що декларує платник, що — податковий агент

| Дохід | Агент | Платник сам |
|---|---|---|
| Українські дивіденди від ТОВ | Утримує | Інформаційно в декларації |
| Українські проценти банку | Утримує | Не обов'язково |
| Українські ОВДП | — | Не оподатковуються (0%) |
| Дивіденди від IBKR / Freedom24 / Wise | Частково (США 15%) | Решту — сам |
| Прибуток з продажу акцій у IBKR | — | Сам |
| Прибуток з продажу акцій через українського брокера | Утримує | Інформаційно |
| Криптовалюта | — | Сам |

## Процес — повний

1. **Січень** наступного року: завантажити всі річні звіти з усіх брокерів.
2. **Лютий**: перерахувати всі операції за курсами НБУ на дати операцій (Excel / бухгалтерське ПЗ).
3. **Березень**: заповнити додатки Ф1 і Ф3.
4. **Квітень**: подати декларацію через кабінет платника з КЕП.
5. **До 1 серпня**: сплатити ПДФО + ВЗ.

## Часті помилки

| Помилка | Наслідок | Правильно |
|---|---|---|
| Не підписав W-8BEN → утримано 30% | Переплата | Підписати; діє 3 роки |
| Курс НБУ — середньорічний | Викривлений фінрезультат | Курс на дату кожної окремої операції |
| Залік утримання у ВЗ | Не визнає ДПС | Залік тільки в ПДФО |
| Задекларував дивіденди Apple під 18% | Переплата | 9% для іноземних дивідендів |
| Пропустив додаток Ф3 | ДПС донарахує при перевірці | Обов'язково для будь-якого іноземного доходу |
| Збитки минулого року зменшує на прибуток цього | Заборонено | Сальдування лише в межах року |
| Забув комісії брокера як витрати | Завищення прибутку | Комісії — в складі витрат на придбання / віднімаються від доходу |

## Обмеження

- Скіл для типових випадків; складні структури (CFC, похідні, опціонні програми від роботодавця) — звертатись до податкового консультанта.
- Для великого обсягу операцій (500+ угод за рік) — обов'язково автоматизоване ПЗ (Koinly, CoinTracker для крипто; Excel + VBA для стоків).
- Оновлення конвенцій — періодично; перевіряти при застосуванні.
