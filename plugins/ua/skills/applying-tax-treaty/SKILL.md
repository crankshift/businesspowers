# applying-tax-treaty

Уникнення подвійного оподаткування для українського резидента, що отримує іноземні доходи (дивіденди, проценти, роялті, заробіток). Застосування конвенцій у межах ст. 170.11.2 ПКУ.

## Ключові джерела

- **ПКУ** (`zakon.rada.gov.ua/laws/show/2755-17`):
  - Ст. 13 — загальний принцип залежності усунення подвійного оподаткування.
  - Ст. 103 — порядок застосування конвенцій.
  - Ст. 170.11.2 — залік іноземного податку з доходів фізосіб.
- **Конвенції з окремими країнами** — за ID на `zakon.rada.gov.ua`.
- **MLI** (Multilateral Convention to Implement Tax Treaty Related Measures, Multilateral Instrument) — ратифікований Україною ЗУ № 2692-VIII від 28.02.2019.

## Принцип роботи конвенцій

Типова структура конвенції:

| Стаття | Тема |
|---|---|
| **Ст. 4** | Резидент (ключова — визначення, до якої країни відноситься платник) |
| **Ст. 6** | Доходи від нерухомого майна |
| **Ст. 7** | Прибутки від підприємницької діяльності (permanent establishment — постійне представництво) |
| **Ст. 10** | Дивіденди — знижені ставки |
| **Ст. 11** | Проценти — знижені ставки |
| **Ст. 12** | Роялті — знижені ставки |
| **Ст. 13** | Доходи від відчуження майна (capital gains) |
| **Ст. 14** | Зарплата — загальне правило «оподатковується країною резидентства працівника» |
| **Ст. 18** | Пенсії |
| **Ст. 21** | Інші доходи (не охоплені попередніми статтями) |
| **Ст. 22 / 23** | Усунення подвійного оподаткування |

## Ставки за основними конвенціями

### США — Ukraine, 1994

[zakon.rada.gov.ua/laws/show/840_009](https://zakon.rada.gov.ua/laws/show/840_009)

| Дохід | Ставка утримання в США | Ставка ПДФО в Україні |
|---|---|---|
| Дивіденди (ст. 10) | **15%** (для фізосіб); 5% (для корпорацій з 10%+ участю) | 9% (ст. 167.5.4 ПКУ) |
| Проценти (ст. 11) | **0%** (повне звільнення) | 18% |
| Роялті (ст. 12) | 10% | 18% |
| Capital gains (ст. 13) | Тільки в Україні (резидент-платник) | 18% + 5% |

**Для отримання знижки 15% замість 30% — підписати форму W-8BEN у брокера (IBKR, Freedom24, Wise Stocks).**

## Механізм заліку

### Формула (ст. 170.11.2 ПКУ)

```
Нарахований_ПДФО_UA = Ставка_UA × Сума_брутто_в_грн
Залік = min(Нарахований_ПДФО_UA, Іноземне_утримання_в_грн)
ПДФО_до_сплати_UA = Нарахований_ПДФО_UA − Залік
ВЗ = Ставка_ВЗ_UA × Сума_брутто_в_грн (БЕЗ заліку!)
```

### Приклад — Apple дивіденди

**Вхідні:**
- Брутто $120 × 41,5 = 4 980 грн.
- США утримало 15% = $18 × 41,5 = 747 грн.

**Розрахунок:**
- Нарахований ПДФО UA (9%): 448,20 грн.
- Залік: min(448,20, 747) = **448,20**.
- **ПДФО до сплати в Україну: 0.**
- ВЗ: 5% × 4 980 = **249 грн**.

**Зайві 747 − 448,20 = 298,80 грн** — **не повертаються** ні США, ні Україною.

### Коли немає чого зараховувати

- Якщо утримання в країні-джерелі = 0% (Ірландія UCITS, ОАЕ для роялті).
- Повний ПДФО + ВЗ сплачується в Україні.

### Обмеження заліку

- **Залік лише в межах українського нарахування.** Якщо США утримало 30% (без W-8BEN) — зарахується тільки 9% (нарахований ПДФО). Зайві 21% — втрачені.
- **Залік лише в ПДФО, не ВЗ.**
- **Залік лише для подачі декларації в Україні з документами.**

## Документи для застосування конвенції

### Довідка про статус резидента UA

Для отримання знижки в країні-джерелі (США, Польща тощо) треба довести, що ти — резидент України.

**Довідка:**
- Видає ДПС України за заявою.
- Форма — затверджена Наказом МФУ № 248 від 12.04.2002.
- Термін — зазвичай 1 рік.
- Через кабінет платника.

### Для США: W-8BEN

**Замість довідки** при малих сумах — через брокера (IBKR, Freedom24, Wise Stocks, Revolut) форма W-8BEN.

**Заповнення:**
- Ім'я, прізвище — латиницею.
- Country of residence: Ukraine.
- Foreign tax identification: РНОКПП (опціонально, але допомагає).
- Treaty Article: 10 (для дивідендів), 11 (проценти), 12 (роялті).
- Withholding rate: 15% для дивідендів.
- Дата, підпис.

Діє 3 роки.

### Для заявки в ДПС на повернення (після переплати)

Якщо в країні-джерелі утримано більше, ніж за конвенцією (неправильно нараховано):
- **Повернення** подається в країні-джерелі (не в Україні).
- Для США: через IRS, форма 1040NR або 8233.
- Для Польщі: через KIS (urząd skarbowy właściwy dla podatku u źródła).

## Документи для декларації в Україні

Для підтвердження заліку в додатку Ф3:

- **1042-S** (США, нерезидент) — стандартна форма від брокера.
- **1099-DIV** (США, дивіденди для резидентів/нерезидентів) — не завжди для UA-резидентів.
- **Виписки брокера** з зазначенням: брутто, утримання, чистий.
- **Офіційна довідка** від іноземного податкового органу (ДПС може вимагати для великих сум).

### Як попросити довідку від IRS

- Form 6166 (Certification of US Residency) — **не** для отримання залежно в UA, це форма USA.
- Для українських цілей залік зазвичай достатньо брокерської виписки з 1042-S.

## Permanent Establishment (ПП, постійне представництво)

**Ст. 5 конвенцій.** Якщо діяльність у країні-джерелі здійснюється через:
- Офіс.
- Філію.
- Склад > 6 міс.
- Представника з правом укладання контрактів.

→ виникає **постійне представництво** → країна-джерело **оподатковує як свої резиденти** (повна ставка, не знижена за конвенцією).

Для фізособи-фрілансера, який віддалено працює — ПП зазвичай не виникає (ст. 7, 14 конвенцій).

## PPT (Principal Purpose Test) — MLI

**Ст. 7 MLI** ввела в усі охоплені конвенції **principal purpose test**:

> Якщо одна з головних цілей будь-якої структури або транзакції була **отримання переваг за конвенцією** — ці переваги **не надаються**.

### Приклад застосування

- Компанія на Кіпрі існує тільки для отримання 5% знижки з українських дивідендів, немає реальної діяльності.
- Податкова (українська або кіпрська) може застосувати PPT → ставка знижена не застосовується → 15% стандарт.

### Нюанс для фізосіб

PPT зазвичай не застосовується до фізосіб, які є резидентами України і отримують дивіденди від іноземних компаній, бо у фізособи немає «штучної структури» для отримання переваг.

## Приклади сценаріїв

### Сценарій 1: Дивіденди Apple

**Вхідні:**
- $120 дивідендів, W-8BEN підписана.
- США утримало 15%.

**Результат:**
- ПДФО в UA: 0 (повний залік).
- ВЗ: 5% × 4 980 = 249 грн.

### Сценарій 2: Дивіденди польського банку PKO

**Вхідні:**
- 1 000 PLN дивідендів.
- PL утримало 15% (стандарт для фізосіб).
- Курс НБУ PLN: 10,5.
- 1 000 × 10,5 = 10 500 грн брутто.
- PL утримало: 150 × 10,5 = 1 575 грн.

**Розрахунок:**
- ПДФО UA 9%: 945 грн.
- Залік: 945 (повний).
- ПДФО до сплати: 0.
- ВЗ: 5% × 10 500 = 525 грн.

### Сценарій 3: Роялті з ОАЕ

**Вхідні:**
- $10 000 роялті від компанії в ОАЕ за ліцензію.
- AE утримало 0% (за конвенцією).

**Розрахунок:**
- Курс: 41,5 → 415 000 грн.
- ПДФО UA 18%: 74 700 грн.
- Залік: 0.
- ВЗ: 5% × 415 000 = 20 750 грн.
- **Разом: 95 450 грн.**

### Сценарій 4: Робота дистанційно на компанію США, зарплата $5 000/міс

**Вхідні:**
- Резидент UA (183+ дні в Україні).
- Працівник US LLC (як contractor).
- Платять $5 000 на IBKR / Wise, без утримання.

**Оподаткування:**
- **За ст. 14 Конвенції US-UA** — зарплата оподатковується **тільки в Україні** (країна резидентства).
- В Україні — **18% ПДФО + 5% ВЗ** = **23%** з $5 000 × 41 = 205 000 грн → 47 150 грн/міс.

**Або:**
- Відкрити **ФОП гр. 3** і отримувати як дохід ФОП → 5% + 1% ВЗ + 1 760 ЄСВ = ~7% ефективно.
- Вимагатиме переукладання договорів (як IC замість W-2).

### Сценарій 5: Україно-польський подвійний резидент

**Ситуація:** живе в Польщі 200 днів/рік, в Україні 100 днів. Центр життєвих інтересів — Польща.

- **Резидент Польщі.** За тайбрейкерами ст. 4 конвенції UA-PL.
- Дивіденди з України (ТОВ на ЗС, 5%) → PL оподатковує + UA утримання 5%.
- Зарплата з PL — PL оподатковує (за шкалою / liniowy).
- Декларація в Україні **не подається** (якщо без українських доходів) або **тільки з українськими джерелами**.

(Для PL-частини — плагін `pl:`.)

## Типові помилки

| Помилка | Наслідок | Правильно |
|---|---|---|
| Не підписав W-8BEN → США утримало 30% | 21% не зараховується → переплата | Підписати W-8BEN |
| Застосовує залік до ВЗ | ДПС не визнає | Залік тільки в ПДФО |
| Подає декларацію без документів про утримання | Залік не проходить | Додавати 1042-S / виписку брокера |
| «Мій друг каже, що не треба нічого декларувати» | Штраф + донарахування при перевірці | Завжди декларувати; залік — автоматично |
| Реєструє компанію на Кіпрі для 5% дивідендів | PPT → DP можуть не визнати | Реальна операційна діяльність, substance |
| Пропустив Польщу-конвенцію — застосував 18% | Переплата | 9% для іноземних дивідендів (у т.ч. польських) |

## Обмеження

- Список конвенцій і ставок у скілі — **орієнтовний**; завжди перевіряти актуальну редакцію (MLI може змінити).
- Для складних структур (холдинги, трастові виплати, subpart F income) — консультант з міжнародного податкового права.
- Для резидентів, що переїхали — перевіряти тайбрейкер-тест у ст. 4 конвенції для визначення резидентства.
