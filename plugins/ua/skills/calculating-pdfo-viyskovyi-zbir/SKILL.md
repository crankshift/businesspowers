# calculating-pdfo-viyskovyi-zbir

Податок на доходи фізичних осіб (**ПДФО**) і військовий збір (**ВЗ**) — два паралельні податки, що сплачуються одночасно з більшості доходів фізособи-резидента України. Загальні ставки — 18% + 5% = **23% на 2026**.

## Ключові зміни 2024–2025

- **До 30.11.2024:** ВЗ = **1,5%**.
- **З 01.12.2024:** ВЗ = **5%** для більшості ставок (ЗУ № 4015-IX від 10.10.2024).
- **З 01.01.2025:** для ФОП 3 групи — окремий ВЗ **1% від обороту** (замість 5% від доходу як у ФОП на загальній системі).

Тому при розрахунках **дата має значення**:
- Події до 30.11.2024 — ВЗ 1,5%.
- Події з 01.12.2024 — ВЗ 5% (загальні ставки); 1% (ФОП 3 гр.).

## Ключові джерела

- **ПКУ** (`zakon.rada.gov.ua/laws/show/2755-17`):
  - Ст. 161 — військовий збір.
  - Ст. 162–163 — платники, об'єкт оподаткування.
  - Ст. 164 — база; ст. 164.4 — курс НБУ для іноземних доходів.
  - Ст. 167 — ставки.
  - Ст. 170 — спеціальні види доходів.
  - Ст. 172–174 — нерухомість, рухоме майно, спадщина.
- **ЗУ № 4015-IX від 10.10.2024** — зміна ВЗ до 5%.

## Ставки ПДФО (ст. 167 ПКУ)

| Вид доходу | Ставка ПДФО | Підстава |
|---|---|---|
| **Загальна** (зарплата, гонорари, оренда, стипендії, пенсії > 10 ПМ) | **18%** | Ст. 167.1 |
| Дивіденди від резидента-платника податку на прибуток (ТОВ на загальній) | **5%** | Ст. 167.5.2 |
| Дивіденди від іноземних компаній | **9%** | Ст. 167.5.4 |
| Дивіденди від неплатника податку на прибуток (ТОВ на ЄП 4 гр., ІСІ поза режимом) | **18%** | Ст. 167.5.1 |
| Спадщина 1-го ступеня від резидента | **0%** | Ст. 174.2.1 |
| Спадщина 2-го ступеня | **5%** | Ст. 174.2.2 |
| Спадщина дальших ступенів / від нерезидента / нерезидентом | **18%** | Ст. 174.2.3 |
| Продаж 1-ї нерухомості року + володіння 3+ роки (житло, резидент) | **0%** | Ст. 172.1 |
| Продаж 2-ї і наступних нерухомостей року, або 3-і роки | **5%** | Ст. 172.2 |
| Продаж нерезидентом / нежитлова | **18%** | Ст. 172.5 |
| Продаж 1-го авто в рік (резидент) | **0%** | Ст. 173.1 |
| Продаж 2-го і наступних авто | **5%** | Ст. 173.2 |
| Проценти за депозитами, облігаціями | **18%** | Ст. 167.5.1 |
| Роялті | **18%** | Ст. 167.5.1 |

## Ставки ВЗ

**Загальна ставка ВЗ з 01.12.2024:** **5%**.

**Винятки:**
- ФОП 3 групи — **1% від обороту** (з 01.01.2025).
- ФОП 1, 2 групи — **10% від ПМ** щомісяця (фіксовано).
- Спадщина 1-го ступеня — **0% ВЗ** (п. 161.2.1).
- Продаж 1-ї нерухомості / авто з 0% ПДФО — **0% ВЗ** (симетрично).

## Формули

### Загальний дохід (зарплата, гонорар, оренда, тощо)

```
ПДФО = 18% × Брутто_дохід
ВЗ = 5% × Брутто_дохід
Чистий = Брутто − ПДФО − ВЗ = 77% × Брутто
```

**Приклад:** зарплата 50 000 грн/міс.
```
ПДФО = 9 000
ВЗ = 2 500
Разом податків = 11 500 (23%)
Чистий = 38 500 грн
```

### Дивіденди (різні ставки)

```
Українські від ТОВ на загальній (5%):
  ПДФО = 5% × Брутто
  ВЗ = 5% × Брутто
  Разом = 10%

Іноземні (9%):
  ПДФО = 9% × Брутто
  ВЗ = 5% × Брутто
  Разом = 14%
  (−) Залік іноземного утримання в межах ПДФО

Українські від ТОВ на ЄП (18%):
  ПДФО = 18% × Брутто
  ВЗ = 5% × Брутто
  Разом = 23%
```

### Спадщина

| Ступінь | Формула |
|---|---|
| 1-й (батьки, діти, подружжя) | ПДФО 0%, ВЗ 0% |
| 2-й (брат/сестра, дідусь/бабуся, онуки) | ПДФО = 5%, ВЗ = 5% (сукупно 10%) |
| Далі / нерезиденти | ПДФО = 18%, ВЗ = 5% (23%) |

### Продаж нерухомості (ст. 172)

```
Сценарій: 1-а продажа року + 3+ роки володіння + житло + резидент
  ПДФО = 0%, ВЗ = 0%

Сценарій: наступні продажі / 2- роки / нерезидент (нежитлова → 18% + 5%)
  ПДФО = 5% × max(Ціна_договору, Оцінка)
  ВЗ = 5% × max(Ціна_договору, Оцінка)
```

Нотаріус утримує при посвідченні угоди.

### Продаж авто (ст. 173)

```
Сценарій: 1-а продажа легкового авто в рік + резидент
  ПДФО = 0%, ВЗ = 0%

Сценарій: наступні / вантажне / нерезидент
  ПДФО = 5% × max(Ціна_договору, Оцінка_за_довідниками)
  ВЗ = 5%
```

## Залік іноземного податку (дивіденди)

Для іноземних дивідендів (ст. 170.11.2 ПКУ):

```
ПДФО_до_сплати_в_Україну = max(0, 9% × Брутто_грн − Іноземне_утримання_грн)
ВЗ = 5% × Брутто_грн (без заліку)
```

### Приклад: Apple дивіденди

**Вхідні:**
- $120 дивідендів від Apple, 15.05.2025.
- США утримало 15% ($18) за W-8BEN.
- Курс НБУ 15.05.2025: 41,5 грн/$.

**Розрахунок:**
- Брутто (грн): 120 × 41,5 = **4 980 грн**.
- Утримано США (грн): 18 × 41,5 = **747 грн**.
- Умовний український ПДФО (9%): 4 980 × 9% = **448,20 грн**.
- Залік: 448,20 ≤ 747 → **ПДФО = 0 грн**.
- ВЗ (5%): 4 980 × 5% = **249 грн** (ВЗ не підлягає заліку!).
- **До сплати в Україну: 249 грн.**

### Приклад: дивіденди з країни з низькою ставкою

**Вхідні:**
- €500 дивідендів від ірландського ETF, 20.07.2025.
- Ірландія утримала 0% (UCITS структура).
- Курс НБУ: 43,5 грн/€.

**Розрахунок:**
- Брутто: 500 × 43,5 = **21 750 грн**.
- Утримано: 0 грн.
- ПДФО: 9% × 21 750 = **1 957,50 грн**.
- ВЗ: 5% × 21 750 = **1 087,50 грн**.
- **Разом: 3 045 грн.**

## Військовий збір для ФОП

### ФОП 1, 2 груп (фіксований)

```
ВЗ_міс = 10% × ПМ_працездатних_01.01
```

Приклад (ПМ = 3 028 грн):
```
ВЗ_міс = 302,80 грн
```

### ФОП 3 групи (з обороту, з 01.01.2025)

```
ВЗ_квартал = 1% × Оборот_квартал
```

Приклад: оборот 300 000 грн.
```
ВЗ_квартал = 3 000 грн
```

### ФОП на загальній системі

**ВЗ — 5% з чистого доходу** (паралельно з ПДФО 18%):
```
Чистий_дохід = Виручка − Витрати (ст. 177.4 ПКУ)
ПДФО = 18% × Чистий
ВЗ = 5% × Чистий
Разом з ЄСВ (22%) = 45% навантаження
```

## Процес розрахунку

### Крок 1. Визначити тип доходу

- Зарплата → загальні 18% + 5%.
- Дивіденд → спецставка.
- Спадщина / дарунок → за ступенем спорідненості.
- Продаж майна → за умовами.
- Оренда → 18% + 5% (або через ФОП).
- Іноземний → 18% + 5% (або 9% + 5% для дивідендів).

### Крок 2. Визначити дату

- До 30.11.2024 — ВЗ 1,5%.
- З 01.12.2024 — ВЗ 5% (з винятками).

### Крок 3. Визначити, чи є податковий агент

- **Є** (роботодавець, ТОВ-виплатник дивідендів, нотаріус при угоді) — агент утримує, фізособа може не декларувати.
- **Немає** (іноземний дохід, орендар-фізособа, приватна спадщина) — платник декларує самостійно до 1 травня + сплачує до 1 серпня.

### Крок 4. Застосувати залік для іноземних дивідендів

За п. 170.11.2 ПКУ — в межах ПДФО (не ВЗ).

### Крок 5. Звести суми

Формат:

| Дохід | Брутто | ПДФО | ВЗ | Утримано агентом | Сплата платником |
|---|---|---|---|---|---|
| Зарплата за 2025 | 600 000 | 108 000 | 30 000 | 138 000 | 0 |
| Дивіденди Apple | 4 980 | 0 (залік) | 249 | 0 | 249 |
| Продаж авто | 800 000 | 40 000 | 40 000 | 80 000 (через МВС) | 0 |

## Соціальна податкова пільга (ст. 169 ПКУ)

Застосовується **лише до зарплати** роботодавцем.

### Стандартна пільга

- **50% × ПМ_працездатних** на 01.01 звітного року.
- Застосовується до однієї (основної) зарплати.
- Ліміт: зарплата ≤ 1,4 × ПМ (округлено).

### Підвищена пільга — 150%

Для:
- одиноких матерів/батьків (з дітьми до 18 р.);
- учасників АТО;
- осіб з інвалідністю III групи.

### Підвищена пільга — 200%

Для:
- учасників ООС з посвідченням;
- осіб з інвалідністю I, II груп.

**Зменшення ПДФО:**
```
ПДФО = 18% × max(0, Зарплата − Пільга)
```

## Типові сценарії

### Сценарій 1: Зарплата 50 000, без пільг

- ПДФО: 18% × 50 000 = 9 000 грн.
- ВЗ: 5% × 50 000 = 2 500 грн.
- Чистий: 38 500 грн.

### Сценарій 2: Зарплата 12 000, стандартна пільга (50% × ПМ = 1 514 грн)

- Пільга застосовується (12 000 ≤ 1,4 × 3 028 ≈ 4 240; НІ — пільга не застосовується, бо ліміт не перевищений... насправді пільга застосовується лише при ЗП ≤ 4 240 грн).
- Припустимо зарплата 4 000 грн → пільга 1 514 → база 2 486 → ПДФО 447,48.
- ВЗ: 5% × 4 000 = 200 грн.

### Сценарій 3: Спадщина від бабусі (2-й ступінь), вартість 500 000 грн

- ПДФО: 5% × 500 000 = 25 000 грн.
- ВЗ: 5% × 500 000 = 25 000 грн.
- Разом: 50 000 грн.

### Сценарій 4: Продаж другого авто в році за 800 000 грн

- ПДФО: 5% × 800 000 = 40 000 грн.
- ВЗ: 5% × 800 000 = 40 000 грн.
- Разом: 80 000 грн (утримує МВС при переоформленні).

### Сценарій 5: Оренда квартири за 20 000/міс

**Варіант A (фізособа):**
- Річний дохід: 240 000.
- ПДФО: 18% × 240 000 = 43 200.
- ВЗ: 5% × 240 000 = 12 000.
- Разом: **55 200 грн/рік** (23%).

**Варіант Б (ФОП гр. 3):**
- ЄП: 5% × 240 000 = 12 000.
- ВЗ 1%: 2 400.
- ЄСВ: ~21 120 (мінімум).
- Разом: **35 520 грн/рік** (≈ 15%).

ФОП вигідніший для оренди значних сум.

## Типові помилки

| Помилка | Наслідок | Правильно |
|---|---|---|
| Застосовує ВЗ 1,5% до доходів після 01.12.2024 | Недоплата | ВЗ 5% з 01.12.2024 |
| Зараховує іноземне утримання у ВЗ | ДПС не визнає | Залік лише в ПДФО (ст. 170.11.2) |
| Декларує дивіденди від Apple за ставкою 18% | Переплата | Ставка 9% (ст. 167.5.4 ПКУ) для іноземних дивідендів |
| Нотаріус утримав — не подає декларацію | Часто це правильно, АЛЕ для другого продажу року — треба декларація | Якщо НЕ перший продаж в рік — декларація обов'язкова |
| Не декларує спадщину 1-го ступеня (0%) | Формально порушення | Подати з 0% |
| Дивіденди через брокера з не-українським утриманням не декларує | Штрафи | Завжди декларувати (з заліком / без) |

## Обмеження

- Ставки можуть змінюватись — **перевіряти поточну редакцію ПКУ**.
- Воєнний збір для конкретних видів доходів — деталі у ст. 161 ПКУ; перевіряти в кожному випадку.
- Соціальна пільга застосовується **роботодавцем**, не самим платником.
- Для складних структур (контрольовані іноземні компанії, CFC) — окремі правила ст. 39² ПКУ.
