# fetching-diia-resources

Скіл — робота з «Дія» (Ministry of Digital Transformation of Ukraine) для реєстраційних дій та електронного підпису.

## Платформи Дія

### 1. Дія — застосунок (iOS, Android)

[diia.gov.ua](https://diia.gov.ua) → завантажити.

**Функції:**
- Електронні документи: паспорт, РНОКПП, свідоцтво про водіння, довідки.
- **Дія.Підпис** — мобільний КЕП через BankID.
- **Дія.ID** — автентифікація по всій державі.
- Заяви в органи: субсидія, допомога ВПО, реєстрація ФОП.

### 2. Дія — веб-портал

[diia.gov.ua](https://diia.gov.ua) — сайт.

**Функції:**
- Вхід через Дія.ID / BankID / КЕП.
- Послуги для громадян і бізнесу.
- Діалог з держорганами.

### 3. Дія.Бізнес

[business.diia.gov.ua](https://business.diia.gov.ua) — окремий портал для підприємців.

**Функції:**
- Інформація про реєстрацію ФОП, ТОВ.
- Грантові програми («єРобота», «Власна справа», «Мікрогрант»).
- Фінансування («Доступні кредити 5-7-9»).
- Бізнес-інструменти, гайди.

### 4. Дія City

[city.diia.gov.ua](https://city.diia.gov.ua) — окремий режим для IT.

## Реєстрація ФОП через Дію

### Крок-за-кроком

1. **Застосунок Дія** → Сервіси → «Для бізнесу» → «Стати ФОП».
2. **Або через веб:** `diia.gov.ua` → Бізнес → Реєстрація ФОП.
3. Автентифікація: Дія.ID (спочатку треба зареєструватись у ній), або BankID (mono, privat24, Ощад, ПУМБ і інші).
4. Заповнення форми:
   - Основний КВЕД (випадіжний список).
   - Додаткові КВЕД.
   - Адреса податкового обліку (пропонується адреса реєстрації з паспорта).
   - Телефон, email.
5. **Вибір системи оподаткування:**
   - «Спрощена система» — група 1 / 2 / 3.
   - Для гр. 3 — 5% без ПДВ або 3% + ПДВ.
   - Або «Загальна».
6. Підпис — Дія.Підписом.
7. Надсилання.
8. Через 1–2 робочі дні — підтвердження від держреєстратора.

### Документи в результаті

- Витяг з ЄДР (у «Мої документи» → «ФОП»).
- Можна завантажити PDF.

## Дія.Підпис

### Що це

Мобільний КЕП (кваліфікований електронний підпис), створений застосунком Дія з використанням:
- Біометричних даних паспорта (для ID-картки з чипом).
- Даних BankID (ключ від банку).

### Генерація

1. Застосунок Дія → Меню → Дія.Підпис.
2. Вибір способу підтвердження: BankID або NFC (приложити ID-картку до телефону).
3. Генерація ключа (автоматично).
4. Встановлення PIN для використання (4 цифри).
5. Готово.

### Термін дії

- **2 роки** з дати видачі.
- Автоматичне оновлення через застосунок (треба повторити процедуру).

### Коли підходить

- Подання електронної звітності (декларації ЄП, ПДВ, ЄСВ, 4-ДФ).
- Підпис угод в електронному вигляді.
- Реєстрація ФОП / ТОВ.
- Зміна даних у ЄДР.

### Коли НЕ підходить

- **Не підтримується 1С:Бухгалтерія** (на момент написання — перевіряти оновлення).
- **Часткове обмеження в M.E.Doc** — перевіряти останню версію.
- Банківські операції — не замінює BankID.

### Альтернативи (якщо Дія.Підпис не підходить)

- **Ключ.України** — безкоштовно для громадян, АЦСК Мін'юсту.
- **Приватбанк** — безкоштовно для клієнтів.
- **Ощадбанк** — безкоштовно для клієнтів.
- **АЦСК «ІСД Мінфіну»** — платне, для юросіб.
- **Апаратні токени** (Алмазний ключ, Кристал-1) — з фізичним носієм.

## Зміни в реєстрації ФОП (форма 11)

**Коли застосовується:**
- Зміна адреси реєстрації.
- Додавання / видалення КВЕД.
- Зміна контактних даних.
- Перехід на нову систему оподаткування.

**Через Дію:**
1. Мої документи → ФОП → Редагувати.
2. Внести зміни.
3. Підпис.
4. Реєстрація — 1 робочий день.

## Закриття ФОП

**Через Дію:**
1. Застосунок / Веб → Бізнес → Припинення ФОП.
2. Автоматичне заповнення форми 11.
3. Підпис.
4. Реєстрація припинення в ЄДР — зазвичай наступного робочого дня.

**Після припинення** — фінальна звітність (див. скіл `closing-fop-checklist`).

## Діа.Бізнес ресурси

### Грантові програми

- **«єРобота»** — гранти для стартапів у прифронтових регіонах.
- **«Власна справа»** — до 250 000 грн для ВПО.
- **«Мікрогрант»** — до 50 000 грн для молоді.
- **«єВідновлення»** — для відновлення бізнесу після обстрілів.

Актуальний перелік — `business.diia.gov.ua/granty`.

### Фінансування

- **«Доступні кредити 5-7-9»** — пільгові кредити з частковою компенсацією відсотків.
- **Портал:** `5-7-9.diia.gov.ua`.

### Консультації

- Онлайн-чат з бізнес-консультантами.
- Довідник порад для початківців.

## Дія City

### Що це

Спеціальний режим для IT-індустрії (ЗУ № 1667-IX від 15.07.2021).

### Переваги для резидентів

- **5% ПДФО + ВЗ** (замість 18% + 5%) для IT-фахівців, оформлених як гіг-контрактори.
- Не сплачується ЄСВ з гіг-контрактів.
- Ставка податку на прибуток компаній — 9% (замість 18%) для резидентів.
- Експортні пільги з ПДВ.

### Вимоги до резидентів

- Компанія в IT-сфері.
- Мінімум 9 працівників/контракторів.
- Середня ЗП ≥ 1 200 євро на особу.
- Частка IT-послуг ≥ 90% виручки.
- Всі перевірки — через портал `city.diia.gov.ua`.

### Реєстр резидентів

- Публічний на `city.diia.gov.ua`.
- Перевіряти статус будь-якого резидента.

## Часті проблеми

### BankID не працює

- Перевірити версію застосунку банку.
- Перевірити, чи підписаний договір з банком про надання BankID.
- Альтернатива — NFC (приложити ID-картку до телефону з NFC-модулем).

### Дія не приймає реєстрацію ФОП

- Помилки у КВЕД → перевірити заборонені для ЄП (`kved-codes-reference`).
- Невідповідність адреси паспорту → оновити паспорт через ДМС.
- Технічна проблема → звернутись у підтримку Дія.

### КЕП застарів

- Оновити через застосунок (якщо 2 роки минули).
- Альтернатива — отримати новий в Ключ.України.

### Витяг з ЄДР не генерується

- Очистити кеш застосунку.
- Перезавантажити.
- Звернутись у підтримку.

## Підтримка

- Телефон: **15-47** (24/7, безкоштовно з України).
- Чат у застосунку.
- Email: info@diia.gov.ua.

## Важливо: зберігання даних

- Дія зберігає копії електронних документів у хмарі держави.
- Для використання — просто показати екран застосунку.
- Офіційний статус — електронний документ має рівну силу з паперовим.

## Обмеження

- Дія — для **громадян України з eID / паспортом-ID**.
- Якщо немає ID-картки — через Ключ.України / ЦНАП.
- Для нерезидентів — Дія не доступна (тільки ЦНАП).
- Мобільний застосунок — працює лише на українському мобільному номері.
