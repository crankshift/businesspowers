# closing-fop-checklist

Повний покроковий чек-лист закриття ФОП в Україні. **Головне правило:** до моменту реєстрації припинення у ЄДР ФОП вважається активним → ЄСВ (мінімум) нараховується автоматично. Чим довше тягнеш з подачею заяви після фактичного припинення діяльності — тим більший борг.

## Ключові джерела

- ЗУ «Про державну реєстрацію…» (`zakon.rada.gov.ua/laws/show/755-15`) — ст. 8, 18 (припинення).
- ПКУ (`2755-17`) — ст. 65, 184, 177.11, 299.
- ЗУ «Про ЄСВ» (`2464-17`) — ст. 4, 6.

## Параметри — перехресне посилання

> Актуальні значення — отримувати через канонічні скіли.
> Якщо скіл не викликаний, використати fallback нижче; попередити: «⚠ Значення [параметр] використано зі станом на [дата].»

| Параметр | Канонічний скіл | Fallback |
|---|---|---|
| ЄСВ / МЗП | `calculating-esv` | 22% × МЗП (мін. 1 760 грн/міс) _(01.01.2026)_ |
| ПДВ / штрафи | `issuing-invoice-ua` | 20% _(01.01.2026)_ |

## Етап 0. Підготовка (за 2–4 тижні до подачі)

- [ ] Вирішити: **закрити повністю** чи **призупинити** (якщо тимчасово — ФОП не призупиняється, але можна знизити активність; ЄСВ все одно йтиме).
- [ ] Перевірити, чи не краще перейти на інший режим (група / загальна) замість закриття.
- [ ] Призначити реальну дату припинення — від неї рахувати все далі.

## Етап 1. Припинити активну діяльність

- [ ] Відмовити / закрити всі відкриті рахунки, договори з клієнтами.
- [ ] Завершити всі виконувані проєкти.
- [ ] Виставити останні рахунки, отримати останні платежі.

## Етап 2. Звільнити найманих (якщо є)

- [ ] Підписати накази про звільнення (ст. 36 або ст. 40 КЗпП).
- [ ] Виплатити остаточний розрахунок (ст. 116 КЗпП: усе — на день звільнення, інакше — компенсація за зволікання).
- [ ] Видати трудові книжки і довідки про зарплату.
- [ ] Подати звіт з ЄСВ за найманих за останні місяці (додаток 5).
- [ ] Повідомити ДПС про звільнення (у межах Об'єднаного звіту).

## Етап 3. Сплатити усі податки до моменту припинення

- [ ] **ЄП** за поточний квартал (пропорційно дням активності).
- [ ] **ЄСВ** за останній квартал.
- [ ] **ПДВ** за поточний / минулий місяць.
- [ ] **ПДФО + ВЗ** (якщо загальна система).
- [ ] **Військовий збір** (1% _(fallback; з 01.01.2025)_ для групи 3).
- [ ] **4-ДФ** за останній квартал (якщо були виплати).

**Перевірка:** кабінет платника `cabinet.tax.gov.ua` → «Стан розрахунків» — має бути **0 або переплата** за всіма податками. Якщо є борг — сплатити, дочекатись оновлення (24–48 год), тоді переходити далі.

## Етап 4. Подання заяви про припинення

### Варіант А: Дія (рекомендований)

- [ ] Застосунок або `diia.gov.ua` → «Для бізнесу» → «Припинення ФОП».
- [ ] Автентифікація Дія.Підписом.
- [ ] Форма 11 заповнюється автоматично з ЄДР.
- [ ] Підпис Дія.Підписом.
- [ ] **Дата реєстрації припинення = наступний робочий день** (зазвичай).

### Варіант Б: ЦНАП

- [ ] Зайти в ЦНАП (будь-який, з 2022 — не тільки за місцем реєстрації).
- [ ] Заповнити форму 11 «Заява про припинення діяльності ФОП».
- [ ] Додати копію паспорта + РНОКПП.
- [ ] Підписати в присутності адміністратора.
- [ ] Розписка про прийняття.
- [ ] Реєстрація в ЄДР — протягом 1 робочого дня.

### Варіант В: через нотаріуса / реєстратора

- [ ] Схожа процедура, платна послуга нотаріуса.

## Етап 5. Отримати витяг з ЄДР про припинення

- [ ] Через Дію → «Мої документи» → ФОП → «Статус: припинено».
- [ ] Зберегти PDF витягу (буде потрібен для фінальних податкових звітів).
- [ ] Дата припинення — фіксується в ЄДР.

## Етап 6. Фінальна податкова звітність

### Для ФОП гр. 3 (або всіх на спрощеній)

**Протягом 40 днів** після дати припинення:

- [ ] **Декларація платника ЄП** з позначкою «довідкова» / «за період до припинення».
  - Заповнити дохід за період від початку кварталу до дати припинення.
  - Розрахунок ЄП + ВЗ 1%.
  - Подати через кабінет з КЕП.
- [ ] **Декларація ПДВ** (якщо платник) — за останній місяць.
  - + **Форма 3-ПДВ** — заява про анулювання реєстрації платника ПДВ.
  - Розрахунок **умовного продажу** за залишками товарів / ОЗ (п. 184.7 ст. 184 ПКУ):

```
Умовний_продаж_ПЗ = 20% × Балансова_вартість_залишку (товари + ОЗ з податковим кредитом)
```

### Для ФОП на загальній системі

**Протягом 30 днів** після припинення:

- [ ] **Декларація про майновий стан і доходи** за звітний період до припинення.
  - Заповнити розділ для ФОП на загальній.
  - Розрахунок ПДФО + ВЗ з чистого доходу.
  - Залишкова амортизація ОЗ — фіксується.

### Звіт з ЄСВ

- [ ] **Річний звіт з ЄСВ** з позначкою «припинено».
  - Якщо припинення до 9 лютого — подавати з повним 2025 роком + вказати дату припинення.
  - Якщо після 9 лютого — звіт уже поданий, донараховувати / виправляти за період.

### 4-ДФ

- [ ] Якщо були виплати фізособам у звітному періоді — подати за квартал припинення.

## Етап 7. Сплатити донараховані суми

- [ ] За фінальною декларацією ЄП — ЄП + ВЗ, якщо ще не сплачено.
- [ ] ПДВ за умовний продаж (якщо анулювали ПДВ з залишками).
- [ ] Залишок ЄСВ (якщо міс. нарахування були більші за сплачені).

## Етап 8. Закрити банківські рахунки

- [ ] Перевести залишки на особистий рахунок фізособи.
- [ ] Закрити рахунок (mono, privat, sense — через застосунок; офлайн — заява у відділенні).
- [ ] Банк автоматично подає **форму 20-ОПП** у ДПС.
- [ ] Перевірити в кабінеті ДПС: рахунок має зникнути зі списку активних.

## Етап 9. Перевірити фінальний статус

Через 30 днів після припинення:

- [ ] **ЄДР** → статус «Припинено».
- [ ] **Кабінет ДПС** → немає активного платника (перевірка за РНОКПП).
- [ ] **Кабінет ПФУ** → статус застрахованої особи, усі борги нульові.
- [ ] **Форма 8-ОПП** (зняття з обліку в ДПС) — має бути видана автоматично.

## Етап 10. Зберегти документи

- [ ] **7 років** — термін зберігання за ст. 44 ПКУ.
- [ ] Усі декларації, квитанції, виписки, договори, акти, ПН.
- [ ] Облікові книги (книга обліку доходів / доходів і витрат).
- [ ] Електронні копії — в хмарі (з шифруванням).

## Головна проблема: ЄСВ «хвостом» після припинення

### Як виникає

1. ФОП вирішує припинити діяльність 01.03.2026, каже «все, закриваюсь».
2. Фактично припинив приймати замовлення 15.03.
3. Заяву про припинення подав 15.05 (через 2 місяці).
4. У ЄДР припинення зареєстроване 16.05.
5. **ЄСВ автоматично нараховано** за березень, квітень, частину травня → **~4 400 грн боргу** (3 × 1 760 _(fallback; станом на 01.01.2026)_ + пропорція).

### Як уникнути

- **Подавати заяву про припинення ОДРАЗУ** (день у день) з моментом фактичного припинення.
- Через Дію — 1–2 робочих дні.
- Через ЦНАП — 1 день.

### Якщо борг уже є

1. Перевірити кабінет ПФУ (`portal.pfu.gov.ua`).
2. Сплатити борг (простіше, ніж сперечатись).
3. Альтернатива — адмінпозов проти ПФУ з доказами фактичного припинення (закриті рахунки, відсутність звітів) — тривале і не завжди успішне.

## Анулювання реєстрації платника ПДВ

### Підстави (ст. 184 ПКУ)

- Припинення підприємницької діяльності.
- Обсяг операцій < 1 млн грн за 12 міс. (якщо був платником добровільно або сталось падіння).
- Недостатня кількість податкових накладних.
- За рішенням ДПС (порушення).

### Форма 3-ПДВ

- Подати **протягом 30 днів** після припинення.
- Електронна форма через кабінет.
- ДПС розглядає 10 робочих днів.
- Рішення — автоматично в кабінеті.

### Умовний продаж

**Ст. 184.7 ПКУ.** При анулюванні — залишки товарів, ОЗ, ... на яких був податковий кредит, вважаються такими, що реалізовано за звичайною ціною → ПЗ.

**Формула:**
```
ПЗ_умовний_продаж = 20% × Справедлива_вартість_залишків
```

**Приклад:**
- ФОП-роздріб, товарні залишки на 100 000 грн (з них ПК у свій час = 20 000 грн).
- При анулюванні: 20% × 100 000 = **20 000 грн ПЗ**.
- Якщо є власний залишковий ПК — можна зачесть; інакше — сплата.

**Не нараховується ПЗ:**
- Якщо платник не використовував ПК при придбанні (тобто купив без ПН).
- Якщо балансова вартість пасивно дорівнює нулю (наприклад, повністю амортизований ОЗ).

## Особливі ситуації

### Закриття ФОП із Дія City-статусом

- Спочатку — відмова від Дія City (окрема процедура через Міністерство цифрової трансформації).
- Потім — припинення ФОП.

### Закриття ФОП при переїзді за кордон

- Не обов'язково (ФОП може бути українським резидентом навіть при фізичному перебуванні за кордоном).
- Якщо став податковим резидентом іншої країни (183+ днів + центр життєвих інтересів) → закрити в Україні і відкрити в новій юрисдикції.
- Подвійне оподаткування — конвенції.

### Закриття через банкрутство

- Звертатись до арбітражного керуючого.
- Окрема процедура за Кодексом з процедур банкрутства.
- Не покрито цим скілом.

### Посмертне припинення

- Спадкоємці подають заяву з свідоцтвом про смерть.
- Документи — через ЦНАП або нотаріуса.
- Борги ФОП → в склад спадщини.

## Часовий план: стандартний сценарій

| День | Дія |
|---|---|
| D-14 | Рішення закрити; фінальна звітність за активний період |
| D-7 | Звільнення найманих (якщо є); виплата останньої ЗП |
| D-1 | Сплата всіх податків; перевірка кабінету |
| D0 | Подання заяви через Дію |
| D+1 | Реєстрація припинення в ЄДР |
| D+2 | Переведення залишків; закриття рахунків |
| D+7 | Перевірка в кабінеті ДПС / ПФУ |
| D+20 | Подання форми 3-ПДВ (якщо платник ПДВ) |
| D+30 | Фінальна декларація (для загальної системи) |
| D+40 | Фінальна декларація ЄП (для гр. 3) |
| D+60 | Перевірка форми 8-ОПП (зняття з обліку) |

## Типові помилки

| Помилка | Наслідок | Правильно |
|---|---|---|
| Подав заяву через 2 місяці після припинення діяльності | ЄСВ за весь період → борг | Подавати **день у день** з рішенням закрити |
| Не подав фінальну декларацію ЄП | Штраф 340 грн _(fallback)_ + можливі донарахування | Подати протягом 40 днів |
| Не подав 3-ПДВ при анулюванні реєстрації ПДВ | ДПС продовжує чекати декларації → штрафи | Протягом 30 днів |
| Забув про умовний продаж → ДПС донарахує при перевірці | Штраф 50% + пеня | Розрахувати самостійно у фінальній декларації ПДВ |
| Закрив рахунок до фінальних платежів | Неможливо сплатити → борги → проблеми зі статусом | Спочатку сплата, потім закриття |
| Відкрив новий ФОП, не закривши борги старого | Борги переходять як на фізособу-боржника | Дочекатись повного закриття → після D+60 відкривати новий |

## Обмеження

- Скіл орієнтовний — для типових випадків.
- При складних ситуаціях (великі залишки ОЗ, незакриті претензії клієнтів, земельні ділянки) — радитись з бухгалтером.
- Податкові перевірки після припинення (до 3 років) — можливі; зберігати документи 7 років.
