# opening-fop-checklist

Повний покроковий чек-лист відкриття ФОП в Україні. Скіл призначений як **готовий список завдань**, що покриває всі етапи від підготовки до першого легального робочого дня.

## Ключові джерела

- ЗУ «Про державну реєстрацію…» (`zakon.rada.gov.ua/laws/show/755-15`) — ст. 18 (реєстрація ФОП).
- ПКУ (`2755-17`) — ст. 65 (облік платників податків).
- Наказ МФУ № 1588 від 09.12.2011 — порядок обліку платників.
- Наказ МФУ № 308 від 19.06.2015 (`z0797-15`) — форма заяви про спрощену систему.
- Постанова КМУ № 69 від 25.01.2023 — порядок послуг через ЦНАП / Дію.

## Параметри — перехресне посилання

> Актуальні значення — отримувати через канонічні скіли.
> Якщо скіл не викликаний, використати fallback нижче; попередити: «⚠ Значення [параметр] використано зі станом на [дата].»

| Параметр | Канонічний скіл | Fallback |
|---|---|---|
| ЄСВ / МЗП | `calculating-esv` | 22% × МЗП (мін. 1 760 грн/міс) _(01.01.2026)_ |

## Канали реєстрації

| Канал | Швидкість | Вартість | Коли оптимально |
|---|---|---|---|
| **Дія (онлайн)** | 1–2 робочі дні | 0 грн | Є eID / Дія.Підпис / BankID |
| **ЦНАП (офлайн)** | 3 робочі дні | 0 грн | Без eID, потрібна консультація |
| **Нотаріус / реєстратор** | 1–3 робочі дні | 500–1500 грн | Паралельні нотаріальні дії |

## Етап 0. Підготовка (за 1–3 дні до подачі)

### Документи

- [ ] Паспорт (ID-картка або книжечка). Перевірити: термін дії, актуальна адреса реєстрації.
- [ ] РНОКПП (ідентифікаційний код; для тих, хто відмовився — відмітка в паспорті).
- [ ] Якщо через ЦНАП — фотокопії паспорта (титульна + адреса) і РНОКПП.
- [ ] Якщо підприємство не є резидентом — особливі документи (посвідка на постійне проживання, довідка про резидентський статус).

### Підбір КВЕД

- [ ] Визначити **основний** КВЕД (те, що даватиме найбільшу частку доходу).
- [ ] Додати **додаткові** КВЕД (все, що потенційно робитимеш).
- [ ] **Перевірити кожен на заборону для ЄП** (п. 291.5 ПКУ). Якщо хоч один заборонений → ЄП неможливий.
- [ ] Скіл `kved-codes-reference` для типових кодів.

### Вибір групи ЄП / загальної системи

- [ ] Визначитись із системою. Скіл `choosing-tax-group` + агент `ua:tax-system-advisor`.
- [ ] Якщо група 3 — визначити **5% без ПДВ чи 3% + ПДВ**.

### Банк

- [ ] Обрати банк (онлайн — mono, Приват24, Sense; офлайн — Ощад, Raiffeisen, fuib).
- [ ] Перевірити комісії за валютні надходження (SWIFT), якщо іноземні клієнти.

### КЕП (кваліфікований електронний підпис)

- [ ] Якщо немає — заздалегідь отримати Дія.Підпис (через застосунок, 5 хв).
- [ ] Альтернатива — Ключ.України (безкоштовно), Приватбанк (для клієнтів), АЦСК (платно).

## Етап 1. Подача заяви

### А) Через Дію

- [ ] Зайти в застосунок **Дія** → «Для бізнесу» → «Стати ФОП».
- [ ] Автентифікація: Дія.ID або BankID.
- [ ] Заповнити:
  - Основний КВЕД.
  - Додаткові КВЕД.
  - Адреса податкового обліку (зазвичай адреса реєстрації).
  - Контактні дані (телефон, email).
- [ ] **Обрати систему оподаткування:**
  - «Спрощена система» → група 1 / 2 / 3.
  - Для групи 3 — обрати 5% або 3% + ПДВ.
  - Або «Загальна» (якщо планується загальна система).
- [ ] Підписати Дія.Підписом.
- [ ] Надіслати.

### Б) Через ЦНАП

- [ ] Звернутись у будь-який ЦНАП (з 2022 — за екстериторіальним принципом).
- [ ] Отримати від адміністратора:
  - **Реєстраційну картку форми 10** (Наказ МФУ № 1588).
  - **Заяву про застосування спрощеної системи** (Наказ МФУ № 308).
- [ ] Заповнити (або заповнити заздалегідь у роздрукованому вигляді).
- [ ] Додати копії паспорта + РНОКПП.
- [ ] Підписати в присутності адміністратора.
- [ ] Отримати розписку про прийняття документів.

### В) Через нотаріуса / реєстратора

- [ ] Звернутись до нотаріуса, який має АРМ реєстратора.
- [ ] Заповнити ті самі форми.
- [ ] Оплатити послуги нотаріуса (500–1500 грн).

## Етап 2. Перевірка реєстрації

### Через 1–3 робочі дні

- [ ] Перевірити **виписку з ЄДР** (через Дію → «Мої документи» → «ФОП»).
- [ ] Альтернативно — [usr.minjust.gov.ua](https://usr.minjust.gov.ua) → пошук за РНОКПП.
- [ ] Переконатись, що в графі «Стан» вказано «Не припинений».
- [ ] Перевірити правильність даних: адреса, КВЕД, дата реєстрації.

### Якщо є помилки

- [ ] Подати заяву про виправлення (форма 11 — зміни).

## Етап 3. Перевірка в ДПС

- [ ] Зайти на [cabinet.tax.gov.ua](https://cabinet.tax.gov.ua).
- [ ] Автентифікація КЕПом.
- [ ] Перевірити:
  - Дата взяття на облік як платника податків.
  - Застосування спрощеної системи (група ЄП).
  - КВЕД у Реєстрі платників ЄП.
  - Адреса ДПІ.
- [ ] Якщо щось не так — подати заяву через кабінет або звернутись до ДПІ.

## Етап 4. Відкриття банківського рахунку

### Через онлайн-банк (mono, Приват24 бізнес, Sense)

- [ ] У застосунку → «Бізнес» → «Стати клієнтом» / «Відкрити рахунок ФОП».
- [ ] Підтвердити дані (автоматично з ЄДР).
- [ ] Обрати валютні рахунки (USD, EUR за потреби).
- [ ] Підписати угоду (через BankID).
- [ ] Дочекатись повідомлення про відкриття (зазвичай за годину).
- [ ] Банк автоматично повідомляє ДПС за формою 20-ОПП.

### Офлайн (Ощад, Raiffeisen, fuib)

- [ ] Звернутись у відділення з паспортом, РНОКПП, випискою з ЄДР.
- [ ] Підписати угоду на розрахунково-касове обслуговування.
- [ ] Отримати IBAN.

### Важливо: **не змішувати** особистий і бізнес-рахунки

- Особистий рахунок — для власних потреб.
- Бізнес-рахунок ФОП — для розрахунків з клієнтами.
- Рух коштів між ними — через виплати на власні потреби (без податків, у межах розрахунків за ЕП / після загальної системи).

## Етап 5. Налаштування інструментів

### КЕП

- [ ] Дія.Підпис — через застосунок (безкоштовно).
- [ ] АЦСК — якщо потрібен для M.E.Doc, 1С або апаратного токена.
- [ ] Резервна копія ключа (на зовнішньому носії; не в хмарі без шифрування).

### РРО/ПРРО (за потреби)

**Обов'язково для:**
- КВЕД 47.xx (роздрібна торгівля) + виручка > 1 млн грн/рік (або без ліміту для деяких КВЕД — перевіряти).
- Побутові послуги населенню (частина КВЕД 95, 96).
- Продаж складної побутової техніки, ліків, ювелірки.

**Не обов'язково для:**
- Послуг юрособам (B2B).
- Експорту послуг.
- IT, консалтингу.

**ПРРО (програмний РРО) — безкоштовна альтернатива:**
- [ ] Завантажити мобільний ПРРО на `tax.gov.ua`.
- [ ] Зареєструвати з КЕПом.
- [ ] Пов'язати з касовою книгою.

### Програми для бухгалтерії (опційно)

- **M.E.Doc** — для електронної звітності, ПН.
- **Sonata** — для ФОП; проста.
- **Fredo Звіт** — безкоштовна.
- **MyDocs** — інтеграція з mono, privat24.

## Етап 6. Перший робочий день

- [ ] Видати перший рахунок / отримати перший платіж.
- [ ] Занести в облік (книга обліку доходів / доходів і витрат для загальної).
- [ ] Розрахувати перший ЄСВ (аванс за перший квартал, до 19 числа місяця після кварталу).

## Приклад хронології для IT-фрілансера (через Дію)

| День | Дія |
|---|---|
| День 0 (пт) | Підготовка: перевірка паспорта, вибір КВЕД, чищення Дія.Підпис |
| День 0 (пт, 18:00) | Подання заяви через Дію |
| День 1 (сб) | Очікування — зазвичай ДПС обробляє у пн |

| День 3 (пн, 18:00) | Отримання першого платежу, фіксація в обліку |

## Важливі нюанси

### Якщо забув заяву про ЄП

- За замовчуванням — **загальна система** → 45% _(fallback; станом на 01.01.2026)_ навантаження.
- Подати заяву про ЄП пізніше: дію з **першого числа кварталу, наступного за поданням**.
- Наприклад, подав у лютому → ЄП з 1 квітня.

### Якщо адреса не відповідає паспорту

- ФОП реєструється за **адресою реєстрації (прописки)**, а не за фактичним проживанням.
- Якщо змінив адресу — спочатку оновити в паспорті / сервісі ДМС.
- Потім — оновити в ЄДР (форма 11).

### Якщо не резидент

- **Нерезиденти не можуть бути платниками ЄП** (п. 291.5.3 ПКУ).
- Лише загальна система або ТОВ.

### Якщо вже був ФОП у минулому

- ФОП припинений → можна відкрити новий у будь-який момент.
- ФОП «заблокований» за борг → спочатку погасити, тоді ДПС зніме блокування.

### Паралельна робота за трудовим договором

- ФОП + зарплата одночасно — **дозволено**.
- Якщо роботодавець сплачує ЄСВ ≥ мін. — ФОП може подати заяву про звільнення від мін. ЄСВ (ст. 7 ч. 6 ЗУ № 2464-17).

### Паралельно ФОП + JDG у Польщі

- Можливо, але обережно:
  - Податковий резидент — лише однієї країни (за місцем 183+ днів або центром життєвих інтересів).
  - Якщо живеш у Польщі > 183 днів → резидент PL → доходи ФОП треба декларувати в Польщі (з заліком українських податків).
  - Плагін `pl:` — для JDG-частини.

## Типові помилки

| Помилка | Наслідок | Виправлення |
|---|---|---|
| Не подав заяву про ЄП → автоматично на загальну | Перший період — 18%+5%+22% = 45% _(fallback)_ | Подати заяву, дочекатись наступного кварталу |
| Обрав заборонений КВЕД | Відмова в ЄП | Виключити з заявки |
| Не відкрив банківський рахунок | Неможливо приймати безготівку | Відкрити не пізніше перших розрахунків |
| Не отримав КЕП | Неможлива електронна звітність | Дія.Підпис за 5 хв через застосунок |
| Не налаштував ПРРО при обов'язковому | Штраф 100–300% необлікованої виручки | Встановити ПРРО до першого розрахунку з населенням |
| Забув про 20-ОПП | Банк подає сам; ФОП має тільки перевірити в кабінеті | Перевірка в кабінеті через 1–2 дні після відкриття |
| Не перевірив актуальність паспортної адреси | Повідомлення ДПС ідуть не туди → пропуск строків | Спочатку оновити паспорт, потім реєструвати ФОП |

## Після першого кварталу (+3 міс.)

- [ ] Подати першу декларацію:
  - Група 3 — квартальна, до 40 днів після кварталу.
  - Групи 1, 2 — нічого поки, до 1 березня наступного року.
- [ ] Сплатити:
  - ЄП (групи 1, 2 — щомісяця; група 3 — після подання декларації).
  - ЄСВ — до 19 числа після кварталу.
  - ВЗ 1% (група 3) — разом з ЄП.

Скіл `reporting-deadlines-ua` — повний календар.

## Обмеження

- Чек-лист орієнтовний; конкретні форми можуть оновлюватись. Перевіряти на `tax.gov.ua` актуальні форми.
- Для складних випадків (перехід з іншої юрисдикції, специфічні КВЕД, наявність нерухомості у статусному обліку) — звертатись до бухгалтера.
